# 💼 Jobable – Job Search Application UI

Jobable is a sleek and responsive job search application built using modern frontend technologies. It focuses on delivering a smooth user experience for browsing and exploring job opportunities.

> ⚡ Currently frontend-only. Backend integration and dynamic features are planned for future updates.

---

## 🚀 Features

- 🔍 Browse job listings
- 🎯 Clean and modern UI
- 📱 Fully responsive design
- ⚡ Fast and optimized performance
- 🎨 Styled using Tailwind CSS + Material UI

---

## 🛠️ Tech Stack

- **React** – Component-based UI
- **TypeScript** – Type safety & better development experience
- **Tailwind CSS** – Utility-first styling
- **Material UI (MUI)** – Pre-built UI components
